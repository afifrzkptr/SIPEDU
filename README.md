# SIPEDU - Sistem Informasi Penduduk Digital Urban

**SIPEDU** adalah aplikasi **Web GIS** yang memvisualisasikan tingkat kepadatan penduduk di Kota Bandar Lampung secara interaktif dan informatif.

🌐 **Demo Aplikasi**: [https://sipedu.vercel.app/](https://sipedu.vercel.app/)

## 📋 Deskripsi Proyek

SIPEDU dibangun untuk membantu memahami sebaran dan kepadatan penduduk di Kota Bandar Lampung berdasarkan data BPS tahun 2024. Sistem ini memanfaatkan teknologi Sistem Informasi Geografis (SIG) untuk menyajikan informasi secara visual, sehingga memudahkan analisis dan pengambilan keputusan dalam perencanaan kota, pelayanan publik, dan pembangunan infrastruktur.

### Fitur Utama
- Peta interaktif kepadatan penduduk per kecamatan
- Klasifikasi tingkat kepadatan (Sangat Padat, Padat, Cukup Padat, Rendah, Sangat Rendah)
- Ringkasan statistik (rata-rata kepadatan, kecamatan terpadat, dll)
- Tampilan responsif dan mobile-friendly

## 🛠️ Teknologi yang Digunakan

- **Frontend**: HTML, CSS, JavaScript, Leaflet.js
- **Backend**: PostgreSQL + PostGIS
- **Pengolahan Data**: QGIS
- **Hosting**: Vercel

## 📊 Sumber Data
- Batas administrasi Kota Bandar Lampung (BPS)
- Data jumlah penduduk dan luas wilayah per kecamatan tahun 2024

## 🎯 Tujuan Proyek
- Memvisualisasikan distribusi kepadatan penduduk di Kota Bandar Lampung
- Memberikan informasi yang berguna bagi pemerintah dan masyarakat
- Menerapkan konsep Sistem Informasi Geografis dalam bidang kependudukan
